Template-driven forms

Template-driven forms are easy to use for small forms and require no code in the typescript file.
That's why they are called "template-driven".

Template driven text
Your input is: {{ templateDrivenText.value }}

Even though template-driven forms don't require binding to the typescript code, it still might make sense to do so. In this case, use two-way binding via [( )].

Template driven number
Your input is: {{ templateDrivenNumber }}

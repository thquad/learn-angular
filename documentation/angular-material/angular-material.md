# Angular Material

## Theme

Add theme to `angular.json`:

```json
  "styles": [
    "src/styles.scss",
    "node_modules/bootstrap-icons/font/bootstrap-icons.scss",
    "@angular/material/prebuilt-themes/indigo-pink.css",
    "src/_theme.scss"
  ],
```

See [_theme.scss](../../src/_theme.scss) for how to setup a custom theme.
